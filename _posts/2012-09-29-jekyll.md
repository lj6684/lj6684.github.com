### 创建一篇Blog
{% highlight shell %}
rake post title="your title here"
{% endhighlight %}

### 启动本地Jekyll服务
{% highlight shell %}
jekyll --server
{% endhighlight %}

### 提交Blog
{% highlight shell %}
git add .
git commit -m 'say something for this post'
git push
{% endhighlight %}
